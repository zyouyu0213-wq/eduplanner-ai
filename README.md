# P2: AI 留学规划方案生成器

把顾问方法论结构化，做成一个 AI 驱动的工具。
输入学生背景，输出个性化的留学规划方案。

---

## 项目基本信息

- **项目名称**：AI Student Planning Assistant
- **当前版本**：v1.0（已上线）→ v2.0（规划中）
- **技术栈**：Next.js + Claude API
- **项目状态**：升级中

---

## 项目背景

作为留学顾问，需要为每个学生制定个性化的申请规划方案。传统方式：
- 耗时长（每个方案 2-3 小时）
- 质量不稳定（取决于当时状态）
- 难以快速迭代
- 知识积累困难

### 用户痛点

**留学顾问**：
- 重复性工作多
- 需要考虑的因素复杂（成绩 / 背景 / 目标 / 预算 / 时间线）
- 难以保持方案一致性

**学生 / 家长**：
- 信息过载，不知从何开始
- 需要个性化建议
- 希望快速得到初步方案

---

## 产品定位

**AI 辅助的留学规划工具** — 不是完全替代顾问，而是：
- 快速生成结构化的初稿方案
- 顾问在此基础上调整优化
- 提高效率，保证质量下限

## 成功指标

### 效率指标
- 方案生成时间：从 2-3 小时 → 目标 20 分钟
- 顾问修改时间：目标 < 30 分钟

### 质量指标
- 方案完整性：包含所有必要元素
- 个性化程度：用户满意度 > 4/5
- 可执行性：建议具体可行

### 用户指标
- 内部使用：至少 3 个顾问采用
- 学生反馈：收集 10+ 份
